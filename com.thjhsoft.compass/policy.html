🛡️ 隐私保护承诺

数字指南针（以下简称"本应用"）高度重视用户的隐私保护。本应用不会收集、存储、上传或分享您的任何个人信息或位置数据。

1. 信息收集与使用

1.1 我们不收集的信息

  • ❌ 不收集个人身份信息（姓名、邮箱、电话等）
  • ❌ 不收集或上传位置数据
  • ❌ 不收集设备标识信息
  • ❌ 不收集使用行为数据
  • ❌ 不使用任何追踪技术
  • ❌ 不包含任何广告或分析SDK

1.2 本地使用的数据

本应用仅在您的设备本地使用以下数据，不会上传到任何服务器：

  • ✅ 传感器数据：加速度计和磁力计数据，用于计算指南针方向
  • ✅ GPS数据：经纬度坐标、海拔、定位精度，仅在设备本地显示

重要说明：所有数据仅在您的设备内存中临时处理和显示，应用关闭后自动清除，不会被保存或传输。

2. 权限说明

2.1 位置权限

用途：获取GPS坐标、海拔和定位精度信息

使用方式：仅在应用运行时使用，数据仅在设备本地显示

是否必需：可选。拒绝此权限不影响指南针核心功能，但无法显示GPS相关信息

2.2 网络权限

用途：仅用于打开使用说明和隐私协议网页

使用方式：通过系统浏览器打开网页链接

是否必需：可选。仅在您点击菜单中的"使用说明"或"隐私协议"时使用

3. 数据安全

✅ 我们的安全措施

  • 所有数据处理均在设备本地完成
  • 不建立任何网络连接传输数据
  • 不使用云存储或远程服务器
  • 应用不包含任何后台服务
  • 开源透明，代码可在GitHub查看

4. 第三方服务

本应用不使用任何第三方服务，包括但不限于：

  • 分析服务（如Google Analytics、Firebase等）
  • 广告服务
  • 云存储服务
  • 社交媒体SDK
  • 推送通知服务

5. 儿童隐私保护

本应用适合所有年龄段用户使用。由于我们不收集任何个人信息，因此不存在儿童隐私泄露风险。

6. 隐私政策变更

如果我们的隐私政策发生变更，我们会在此页面更新，并更改顶部的"最后更新日期"。重大变更会在应用内通知用户。

⚠️ 重要提示

本隐私政策仅适用于数字指南针应用本身。当您点击链接访问外部网页时，这些网站可能有自己的隐私政策，请注意查看。

🛡️ Privacy Protection Commitment

Digital Compass (hereinafter referred to as "the App") highly values user privacy. This App does not collect, store, upload, or share any personal information or location data.

1. Information Collection and Use

1.1 Information We Do NOT Collect

  • ❌ No personal identification information (name, email, phone, etc.)
  • ❌ No location data collection or upload
  • ❌ No device identification information
  • ❌ No usage behavior data
  • ❌ No tracking technologies
  • ❌ No advertising or analytics SDKs

1.2 Data Used Locally

The App only uses the following data locally on your device and does not upload to any server:

  • ✅ Sensor Data: Accelerometer and magnetometer data for calculating compass direction
  • ✅ GPS Data: Latitude/longitude coordinates, altitude, location accuracy, displayed locally only

Important Note: All data is temporarily processed and displayed in device memory only. Data is automatically cleared when the app closes and is never saved or transmitted.

2. Permission Details

2.1 Location Permission

Purpose: Obtain GPS coordinates, altitude, and location accuracy

Usage: Only used while app is running, data displayed locally only

Required: Optional. Denying this permission does not affect core compass functionality but GPS-related info won't be displayed

2.2 Internet Permission

Purpose: Only for opening user guide and privacy policy web pages

Usage: Opens web links through system browser

Required: Optional. Only used when you click "User Guide" or "Privacy Policy" in menu

3. Data Security

✅ Our Security Measures

  • All data processing is done locally on device
  • No network connections for data transmission
  • No cloud storage or remote servers
  • No background services
  • Open source and transparent, code available on GitHub

4. Third-Party Services

This App does NOT use any third-party services, including but not limited to:

  • Analytics services (e.g., Google Analytics, Firebase)
  • Advertising services
  • Cloud storage services
  • Social media SDKs
  • Push notification services

5. Children's Privacy

This App is suitable for all ages. Since we don't collect any personal information, there is no risk of children's privacy leakage.

6. Privacy Policy Changes

If our privacy policy changes, we will update this page and modify the "Last Updated" date at the top. Significant changes will be notified within the app.

⚠️ Important Notice

This privacy policy applies only to Digital Compass app itself. When you click links to visit external websites, those sites may have their own privacy policies.

7. Contact Us

If you have any questions or suggestions about this privacy policy, please contact us via:

  • 📧 GitHub: Submit an Issue in the project repository
  • 🌐 Project Homepage: https://github.com/yourusername/compass

8. Your Rights

Since this App does not collect any personal data, you don't need to worry about data access, modification, or deletion. You can:

  • Revoke location permission at any time
  • Uninstall the app at any time; all local temporary data will be automatically cleared
  • Review open source code to verify privacy protection measures
