User Manual

How to Play

Chess is a board game for two players. It is played on a square board, made of 64 smaller squares, with eight squares on each side.

Game Modes

  • Player vs Player: Play against a friend on the same device.
  • Player vs AI: Challenge the computer with different difficulty levels.

Features

  • Undo: Revert your last move.
  • Hints: See possible moves for the selected piece.
  • Records: Save and replay your games.
